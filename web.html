BOOM BAP

Ember + JavaScript + SCSS


Boom Bap is a drum machine where a user can hover over the drum pads to sound a beat, create a drum loop with kick and snare or make a loop with the drumpad of their choosing.

Created by Joseph Arnstein, Molly Curtin, Kevin Mattison & Chris Houghton

Eames Office

SCSS + HTML


A responsive website for the Eames Office based on the design of Andersson-Wise's current website.

Folkways

Ember + Firebase + SCSS


Folkways is a site where users can rate their favorite albums from the Smithsonian Folkways archive, create a collection of their favorites and add releases to the site.

MoSK

Ember + Firebase + SCSS


MoSK is an online marketplace for the modern art collector. The user can view individual pages for the various pieces, as well as add pieces to their cart and see the total cost. The administrator of the site will be able to add new art pieces, as well as update and delete entries where necessary.

Created by Molly Curtin & Oskar Radon

CRATE

SCSS + PHP + Discogs API


CRATE is a responsive web application for collecting and organizing your personal record collection. We wanted to take the ideas and database of discogs.com and create a more intuitive and user-friendly UI for viewing and accessing your record collection.

Created by Molly Curtin, Eddie Duro, Lauryn Davis, Jeff Seymour, Yvonna Contreras & Erik Tolentino
