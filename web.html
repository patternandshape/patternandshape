Coroflot + Core77

SCSS, HTML, Sketch/Adobe Illustrator + InVision


While working as an Interaction Designer as part of the Core77 + Coroflot family, I've designed the UI/UX for several important pages, as well as icon sets and navigational menus on the sites. Some of the projects are live and many are in progress.

Most recently, I redesigned our login page as seen above. Below is an icon set I designed for one of our sales pages and to the right is a back-end dashboard for screening out unfit applicants to the site.

Another recent project was the design of our 404 Page. I thought it could use a bit of humor so I implemented a simple CSS animation to make the eyes move.

I'm currently working on a redesign of the navigation system for Core77 as well as a new design for our talent profiles on Coroflot.





Visible Cloaks

SCSS, HTML, Sketch + InVision


For this project, I designed and developed a responsive website for the band Visible Cloaks which houses their videos and "Valve", an interactive digital environment by Brenna Murphy set to their music. Discography and event space to follow.

BOOM BAP

Ember + JavaScript + SCSS


Boom Bap is a drum machine where a user can hover over the drum pads to sound a beat, create a drum loop with kick and snare or make a loop with the drum pad of their choosing. We made this app during Group Project week at Epicodus. I was in charge of the overall design of the drum machine and built it using Ember and SCSS.

Eames Office

SCSS + HTML


A responsive website for the Eames Office based on the design of Andersson-Wise's current website. This was an independent project I made at Epicodus whose challenge was to re-create their site with SASS and without utilizing any front-end frameworks.

Folkways

Ember + Firebase + SCSS


Folkways is a site where users can rate their favorite albums from the Smithsonian Folkways archive, create a collection of their favorites and add releases to the site.

MoSK

Ember + Firebase + SCSS


MoSK is an online marketplace for the modern art collector. The user can view individual pages for the various pieces, as well as add pieces to their cart and see the total cost. The administrator of the site will be able to add new art pieces, as well as update and delete entries where necessary.

Created by Molly Curtin & Oskar Radon

CRATE

SCSS + PHP + Discogs API


CRATE is a responsive web application for collecting and organizing your personal record collection. We wanted to take the ideas and database of discogs.com and create a more intuitive and user-friendly UI for viewing and accessing your record collection. We made this app during Group Project Week at Epicodus and I was the project manager and front-end design lead. The front-end of this project was built with SCSS and JavaScript while our back-end team was working in PHP utilizing the Discogs API.

Arctic Wild

SCSS + HTML


A replica of the website for Arctic Wild, a wilderness guiding company based in Alaska. This was an independent exercise utilizing Flexbox and CSS transitions.

Superstudio

SCSS + Bourbon + HTML


Superstudio was an architecture firm, founded in 1966 in Florence, Italy that was a major part of the Radical architecture movement of the late 1960s. For this independent project I created a responsive blog template for Superstudio based on the RO LU blog.

GitHub User Search

jQuery + GitHub API + SCSS


GitHub User Search facilitates finding a user's list of repositories on GitHub. For this independent project I utilized the GitHub API to enable searching a user's repositories within the site.

Variable Pizza

JavaScript + BDD + CSS


Variable Pizza is a web application where a user chooses a pizza size and toppings to their liking and receives a total cost for said pizza. I created this site for an independent project to demonstrate using Mocha and Chai for BDD testing along with JavaScript.

Gegessen

Angular 2.0 + TypeScript + CSS


Gegessen is a small Angular 2.0 app where a user can track their meals, which happen to be very German. The user of this app can log each food they eat with some details about the meal and number of calories for that portion. They will be able to view and edit the list of the foods they've entered, click on each meal for more information and sort their meals by healthy and unhealthy foods.
